⛓️ Chain Rule in Action

The chain rule allows us to compute gradients for nested functions:

∂L/∂w = ∂L/∂y × ∂y/∂z × ∂z/∂w

Where:

  • L = Loss function
  • y = Network output
  • z = Weighted sum before activation
  • w = Weight parameter
